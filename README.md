## Local Setup
```sh
$ git clone https://github.com/julinha13/React-hooks-calculator.git
```

## Install dependences
```sh
$ npm install
```
## Run Locally 
```sh
$ npm start
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
